Plugin WordPress pentru căutarea și comanda pieselor de schimb prin API-ul EURAS/ASWO EED.

== Description ==

ASWO Parts Search & Order conectează magazinul tău WordPress cu API-ul EURAS EED pentru a permite căutarea și comanda pieselor de schimb direct din site.

**Funcționalități principale:**

* Căutare piese de schimb după număr articol, denumire sau model aparat
* Căutare aparate compatibile
* Coș de cumpărături integrat
* Plasare comenzi directe prin API
* Panou de administrare complet
* Autocomplete pentru câmpul de căutare
* Design responsiv

**Shortcode-uri:**

* `[aswo_parts_search]` - Afișează formularul de căutare și rezultatele
* `[aswo_cart]` - Afișează coșul de cumpărături și formularul de comandă

== Installation ==

1. Încărcați folderul `pluginaswo` în directorul `/wp-content/plugins/`
2. Activați plugin-ul din meniul 'Plugins' al WordPress
3. Mergeți la Settings > ASWO Settings și introduceți credențialele API
4. Creați o pagină pentru căutare și adăugați shortcode-ul `[aswo_parts_search]`
5. Creați o pagină pentru coș și adăugați shortcode-ul `[aswo_cart]`
6. Configurați paginile în setările plugin-ului

== Frequently Asked Questions ==

= De unde obțin credențialele API? =

Credențialele API (Customer ID, Login, Password) se obțin de la EURAS/ASWO la înregistrarea unui cont de reseller accounts.

= Ce versiune PHP este necesară? =

Plugin-ul necesită PHP 7.4 sau mai nou.
