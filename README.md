# Plugin ASWO - WordPress Plugin

Plugin WordPress pentru căutare și comandă piese de schimb prin API-ul EURAS/ASWO EED.

## Funcționalități

- Căutare piese de schimb
- Căutare aparate compatibile
- Coș de cumpărături
- Plasare comenzi
- Panou admin pentru configurare API

Bazat pe documentația EURAS EED: https://shop.euras.com/admin/Dok/eed-doku-eng.php
